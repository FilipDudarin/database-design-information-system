# Database Design – Manufacturing Information System

This repository contains a team university project focused on designing a database schema for an information system that supports the operations of a metal manufacturing company.

## Project overview

The project included:

* business domain analysis
* requirements specification
* identification of system users
* use case modeling
* database schema design
* preparation of project documentation

The main goal was to design a database structure that supports important business processes and enables better management of documentation, resources, and operational data.

## Methods and tools

* requirements analysis
* use case modeling
* EER modeling
* database design
* information system analysis

## Repository contents

* project documentation
* database design materials
* diagrams and supporting models
